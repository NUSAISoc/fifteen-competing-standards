# fifteen-competing-standards
High-level bindings and algorithm implementations to decisively solve AI/ML/DL problems

## Installation

We assume the use of `pyenv`/`uv` for Python version management.

Please for the love of everything that is beautiful, do **NOT** use `conda`.

```bash
python -m pip install -U pip
python -m pip install -e "."
```
